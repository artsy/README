## Practice Information

Web Practice meets bi-weekly to share and discuss trends across Artsy’s web space, raise challenges we’re seeing, and align on the direction of our frontend work.

### Resources

- [Web Practice upcoming meetings, notes, and resources](https://www.notion.so/artsy/Web-Practice-bfe55e2c614a4c0eae493b7830622843)

### Team Goals

- Build and maintain high-quality, user-facing applications.
- Support product goals through thoughtful use of web technologies.
- Prioritize productivity and developer happiness.
- Collaborate on design system alignment and growth.
- Share best practices across the web team.

### Our Technologies

- **TypeScript** - Typed superset of JavaScript
- **React** - Component-based JavaScript library
- **Relay** - GraphQL client for efficient data fetching and state management
- **GraphQL** - Query language for APIs
- **Easy Peasy** - State management library built on Redux
- **Styled Components** - CSS-in-JS library for styling React components
- **Storybook** - UI component development and documentation
- **Jest** - JavaScript testing framework
- **Testing Library** - User-centric testing utilities for React components
- **Playwright** - End-to-end testing framework
- **Yarn** - Package management
- **Visual Studio Code** - Code editor and development environment
- **Stripe** - Payment processing platform for online transactions
- **Unleash** - Feature flag management system
